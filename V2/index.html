🚀 Comment Participer ?

Rejoindre PeaceChain est simple et gratuit. Suivez ces étapes pour devenir acteur du changement :

1

Téléchargez le nœud

Clonez le dépôt GitHub ou téléchargez l'archive ZIP. Installez PHP 7.4+ et les dépendances nécessaires.

git clone https://github.com/votre-compte/peacechain.git
2

Lancez votre nœud

Configurez votre nœud pour rejoindre le réseau décentralisé. Votre ordinateur devient un gardien de la paix numérique.

php node.php --start
3

Créez votre première application

Utilisez nos API et SDK pour développer des applications innovantes sur PeaceChain. Documentations et exemples disponibles.

4

Contribuez à la communauté

Partagez vos créations, aidez les nouveaux venus, participez aux décisions via notre système de gouvernance décentralisé.

5

Propagez la paix

Organisez des événements, créez du contenu éducatif, inspirez d'autres personnes à rejoindre le mouvement.
